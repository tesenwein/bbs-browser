# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in BBS Browser, please report it
privately rather than opening a public issue.

Use GitHub's [private vulnerability reporting](https://github.com/tesenwein/bbs-browser/security/advisories/new)
feature, or open a draft security advisory. Please include:

- A description of the vulnerability and its impact
- Steps to reproduce
- Any relevant logs or proof-of-concept code

We aim to acknowledge reports within a few days and will keep you updated as
the issue is triaged and fixed.

## Supported Versions

Only the latest release on `main` receives security fixes.
